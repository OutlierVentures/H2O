# H2O: Haja to Ocean

Machine learning in Nautilina with OrbitDB.

### Requirements

- Docker, Docker Compose
- Node.js, Yarn, Angular CLI
- Python, pip

### Architecture

![Architecture Diagram](/doc/OceanHaja.png)
